Slideshow plugin - AvineJS

Slideshow permet de générer des Carrousels animés avec différentes méthodes de navigation, mais également une navigation en onglets (Tabs).

Le code HTML

Le code HTML, est une simple liste non-ordonnée contenant les différents slides. Le Slideshow Javascript a été conçu sur le principe d'amélioration progressive. N'hésitez-pas à désactiver Javascript dans votre navigateur à tout moment, pour tester le rendu en mode dégradé (qui n'est pas si dégradé que ça...).

<ul id="slideshow-x">
	<li>[Content]</li>
	<li>[Content]</li>
	<li>[Content]</li>
</ul>

Notez que l'on peut en fait appliquer le slideshow à n'importe quelle autre structure HTML. Le principe est simple : le slideshow est appliqué à tous les enfants directs du nœud racine ciblé.

Le code Javascript

Voici le code Javascript permettant d'appliquer le Slideshow à une partie du document HTML.
Hum... Les habitués de jQuery y verront un air de déjà vu. A noter que la méthode "slideshow" est en réalité une extension et ne fait donc pas partie du "core" du framework.

<script>
// Attendre que le DOM soit construit
$.dom.ready(function(){

	// Appliquer le slideshow à l'élément dont l'id vaut "slideshow-1"
	$('#slideshow-1').slideshow();

});
</script>

Un rendu avancé avec mise en forme css...

Appliquons maintenant le slideshow à la liste dont l'id vaut : "slideshow-2". Cette fois, quelques comportements intéressants sont ajoutés. On commence au deuxième slide et la transition est animée. Le défilement des slides est automatique mais s'interrompt lorsque la souris passe au dessus du Slideshow. Notez que l'apparence de la barre de navigation est adaptée en renseignant dans l'instance le paramètre navigCss: 'demo-slides-navig'. L'apparence des slides quant à elle, est obtenue simplement en appliquant la classe css "demo-slides-container" à chaque balise <li>.

<script>
$.dom.ready(function(){

	$('#slideshow-2').slideshow({
		start: 1,
		fade: 250,
		autoSlide: 4000,
		autoStop: true,
		navigCss: 'demo-slides-navig',
		textPrev: 'Précédent',
		textNext: 'Suivant',
		textResume: 'Reprendre'
	});

});
</script>

Et en bonus... un classique du genre

Dans les exemples précédents, les éléments de la barre de navigation étaient insérés dans une balise "fourre-tout", ajoutée automatiquement au-dessus où en-dessous des slides. Cette fois, seul le "compteur" est conservé, alors que les boutons "précédent" et "suivant" ne sont pas affichés. Les boutons des "ancres" sont quant à eux insérés dans une balise spécifiée et dont la classe vaut : "demo-slides-anchors".

<script>
$.dom.ready(function(){
	$('#slideshow-3').slideshow({
		fade: 250,
		autoSlide: 3000,
		autoStop: true,
		navigCss: 'demo-slides-navig',
		navigContainer: {
			prev: false,
			next: false,
			counter: true,
			anchors: '.demo-slides-anchors'
		}
	});
});
</script>

Et pour finir, une navigation en onglets...

Jusqu'à présent, les boutons des "ancres" étaient générés automatiquement. Cette fois, ils sont fournis en dur dans la structure html de la page. Ainsi, il est possible de définir un contenu personnalisé pour chaque bouton...

<script>
$.dom.ready(function(){
	$('#slideshow-4').slideshow({
		fade: 300,
		autoSlide: 2500,
		autoStop: true,
		autoHeight: 200,
		navigContainer: {
			prev: false,
			next: false,
			counter: false,
			anchors: '#slideshow-4-anchors'
		}
	});
});
</script>

Et encore un pour la route, le carrousel en images...

La particularité cette fois, c'est que nous devons attendre le chargement complet des images avant de rendre le carrousel. Cela en raison que les dimensions des images ne sont pas renseignées dans le code HTML. En conséquence les images doivent avoir été chargées pour que la hauteur du slideshow soit correctement calculée. Pour cela, il faut fixer à true le second paramètre de la méthode ready. 

<script>
$.dom.ready(function(){
    $('#slideshow-5').slideshow({
        fade: 600,
        autoSlide: 2500,
        autoStop: true,
        navigPos: 'bottom',
        navigCss: 'demo-slides-anchors',
        navigContainer: {
            anchors: true
        }
	});
}, true); // Wait for resources laoded
</script>
