loading - jQuery plugin

loading permet d'afficher un retour visuel au chargement de données asynchrone. Ce plugin est basé sur un générateur de code simplifiant la création de plugins jQuery.

Introduction

Le code suivant montre la syntaxe de base pour invoquer le plugin sur un object jQuery.

<div class="test"> <!-- content --> </div>
<div class="test"> <!-- content --> </div>

<script> $(".test").loading(); </script>
